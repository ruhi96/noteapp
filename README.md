## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open browser to: **http://localhost:3000**

## API Endpoints

- `GET /api/notes` - Get all notes
- `POST /api/notes` - Create a new note
